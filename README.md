Проект по созданию веб-приложения с встроенным ИИ-помощником

Задача: создать веб-мессенджер, в котором люди, разговаривающие на разных языках, смогли бы беспрепятственно обмениваться информацией и общаться благодаря встроенному ИИ-помощнику, задачей которого будет перевод текста на одного языка на другой (выбор языка, на который будет осуществлён перевод, должен оставаться за пользователем).

Весь проект будет преимущественно делаться на Linux Fedora, возможна работа с Windows 10.
